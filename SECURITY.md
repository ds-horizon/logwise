# Security Policy

We take security seriously. Please report vulnerabilities privately and responsibly.

## Reporting a Vulnerability

- Email: onkar.sarvade@dream11.com
- Acknowledgement target: within 24 hours
- Please do not open public issues for security reports

Include:
- Affected version/commit
- Reproduction steps or proof-of-concept
- Impact assessment and any mitigations

## Response Process

- We will validate the report and provide status updates
- Security fixes are prioritized and released as soon as validated
- Fixes will be included in a patched release and noted in release notes

## PGP/GPG

If you require encrypted communication, request our PGP key in your initial email.
